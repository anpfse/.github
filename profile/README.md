# AI Native Project Faster Starter Ecosystem (ANPFSE)

## 愿景（Vision）

AI Native Project Faster Starter Ecosystem（ANPFSE）旨在构建一个覆盖 AI 时代软件工程全生命周期的脚手架生态系统。

目标不是简单地维护一批 Boilerplate，而是建立：

> AI 原生开发时代的软件工程基础设施标准库。

让开发者能够快速创建符合最佳实践的项目，并随着业务发展逐步演进到企业级架构。

---

# 核心理念

借鉴解析几何中的思想：

```text
点（Point） → 面（Plane） → 体（Solid） → 生态（Ecosystem）
```

## 第一阶段：点（Point）

提供单个应用类型的标准模板。

例如：

- FastAPI
- Next.js
- Flutter
- Tauri
- MCP Server

解决：

```text
如何快速启动项目
```

---

## 第二阶段：面（Plane）

覆盖所有主流应用类型。

```text
Backend
Frontend
Mobile
Desktop
AI
Infra
Data
Security
```

解决：

```text
如何快速启动任何类型的项目
```

---

## 第三阶段：体（Solid）

在模板基础上增加工程能力。

例如：

```text
Starter
 ↓
Standard
 ↓
High Concurrency
 ↓
High Availability
 ↓
Security
 ↓
Enterprise
```

解决：

```text
如何从一个简单项目逐步演化为企业级系统
```

---

## 第四阶段：生态（Ecosystem）

形成完整的软件工程生态：

```text
标准
 ↓
模板
 ↓
生成器
 ↓
平台
```

例如：

```bash
uvx create-anpfse-app

npx create-anpfse-app
```
